Mt4j-extensions

Extensions to the MT4J Framework providing generic, stylable components for user interface.

mt4j-extensions is a Java-based framework which enables users to create multi-touch capable user interfaces. By providing basic shapes and content types, it enables users to create sophisticated touch-based UIs that are completely configurable - by the developer. Mt4J provides a rich set of standard multi-touch gestures that can easily be used with all the components provided with the framework

Yet the components that are provided, while being technically sophisticated and feature-rich, can not be seen as usable by visual artists and user interface prototypers. It requires a vast amount of code to change the style and appearance of elements. My goal is to create higher level visible components, that can be intuitively used as building blocks in multi-touch applications.
